# 🎬 Movie Recommendation System

## 📌 Overview
This is a content-based movie recommendation system built using Machine Learning and Streamlit.

## 🚀 Features
- Recommends similar movies
- Displays movie posters
- Shows ratings and release year
- Uses TMDB API

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 🌐 Live App
https://movie-recommender-anjali.streamlit.app

## 📂 Dataset
Based on movie metadata and similarity matrix.
